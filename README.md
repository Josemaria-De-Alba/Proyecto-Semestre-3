Este proyecto se basa en paginas para ver el porcentaje de uso de personajes como esta:

https://apexlegendsstatus.com/game-stats/legends-pick-rates

Mi codigo es una manera de organizar datos como en esta pagina, esta dise√±ada para juegos en linea, recibe el nombre de un personaje/arma y su nombre o identificador y los acomoda para que el usuario pueda ver estos datos y vea si alguno de estos es demasiado devil o demasiado fuerte atravez de su uso.
